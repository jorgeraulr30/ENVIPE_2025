# ENVIPE_2025
Estudio longitudinal de victimización y percepción de seguridad en Querétaro con ENVIPE 2020-2025. Reporte en R Markdown, gráficas y datos municipales.
"reporte_v6.Rmd" debe ser compilado necesitando únicamente los archivos "resultados_municipios_YYYY.csv" (YYYY={2020, ..., 2025}). La ruta debe ajustarse a la ubicación de los archivos. A su vez, estos mismos son generados, respectivamente, por los archivios "YYYY_municipales.R" y requieren correrse cada uno en la carpeta donde se encuentren
